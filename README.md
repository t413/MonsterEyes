###Make a blinking LED ring with shifty monster eyes

Using the [16-pixel LED ring from Adafruit](http://www.adafruit.com/products/1463), wire up a microcontroller ([I used a trinklet](http://www.adafruit.com/product/1501)) and a battery to give yourself a third eye (or two).
